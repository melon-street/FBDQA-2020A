# 量化交易体系的开启姿势
## 为什么要量化
**凯利公式**  
f^* = (bp-q)/b  
其中  
f^* = 投注金额占总资金的比例  
p = 获胜的概率  
q = 失败的概率, q=1-p  
b = 赔率  
**FBDQA对于量化交易的定义**  
通过系统性方法和程序化手段，进行交易模式发现及交易策略构造，并借助计算机系统进行验证和执行的交易过程。
