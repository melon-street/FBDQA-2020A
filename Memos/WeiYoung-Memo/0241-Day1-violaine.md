# 量化交易体系的开启姿势
## 为什么要量化
**凯利公式**  
f^* = (bp-q)/b  
其中  
f^* = 投注金额占总资金的比例  
p = 获胜的概率  
q = 失败的概率, q=1-p  
b = 赔率  
**FBDQA对于量化交易的定义**  
通过系统性方法和程序化手段，进行交易模式发现及交易策略构造，并借助计算机系统进行验证和执行的交易过程。

# Git使用
## Git
- Git是一套协议，实现了分布式版本管理  
- Git的核心概念Repo仓库及其管理
- Git上项目的操作步骤：
    - Repo Dir，建仓库
    - Add，加文件
    - Commit，提交创建或修改Commit Change
    - Push Remote,各成员反馈意见
    - Merge，编辑汇总、合并等
    - Release，合并发行
- http://rogerdudler.github.io/git-guide/
